# VIP캠핑장 마케팅 상권 분석

## 프로젝트 개요
양주장흥VIP캠핑장 중심 반경 20km 경쟁 캠핑장 상권 분석 + 지도 대시보드

## 데이터 소스
- 카카오 지도 API (REST key: golf_settings.json 참조, mstick-plus 프로젝트)
- 캠핏 API (Playwright CF 우회 필요)
- 카카오 리뷰/평점

## 수집 완료 데이터
- `kakao_camps.json` — 45개 캠핑장 (위치/전화/거리)
- `camfit_matched.json` — 캠핏 매칭 5개 (가격/정책)
- `kakao_reviews.json` — 리뷰/평점
- `merged_camps.json` — 통합 데이터
